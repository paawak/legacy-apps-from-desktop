This is a beta-version and still testing is going on and bugs abound. All bug reports can be made at: paawak@gmail.com. Any constructive criticism are always welcome.

Bhasha is an utility to write in Bangla, Devnagari(Hindi) or English. Also it allows users to generate an HTML file and JPEG image from the Bhasha-pad, as is, so that it can be ported easily by e-mails.

Bhasha, being coded in Java, is platform-independent. To run Bhasha, you have to have JavaRunTimeEnvironment1.1.2(jre1.1.2) or higher. The latest version of jre can be dowloaded at: http://www.java.com/en/download/manual.jsp. Select the appropriate operating system and proceed according to the given instructions.

This is completely free, but not strictly OpenSource. However, developers who wish to have my source code can contact me at: paawak@gmail.com. But modifications to the code must be reviewed by me before incorporation.
