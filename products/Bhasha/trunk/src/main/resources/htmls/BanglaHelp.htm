Rules for typing in Bhasha

Typing is case sensitive - 'S' and 's' are different.
স্বরবর্ণ  গুলির  ক্ষেত্রে,  ছোট হাতের English হরফ, মানে small letters ব্যাবহার করা হয়েছে ৷
যেমন 'স' লিখতে হলে 's' type করুন ৷ আবার 'স্' লিখতে হলে 'S', মানে SHIFT টিপে 's' type করুন ৷ আপনি যদি 'SolaimanLipi.ttf' ব্যবহার করেন, তাহলে যুক্তাক্ষরের জন্য কোন অসুবিধা হবেনা ৷
ব্যাঞ্জনবর্ণ গুলির ক্ষেত্রে, ছোট হাতের English হরফ মাত্রা গুলির জন্য ব্যবহার  করা হয়েছে ৷
যেমন 'কি' লিখতে গেলে 'ki' type করুন  ৷ 'কই' লিখতে গেলে 'kI' type করুন ৷

To break the continuity between two characters type: CTRL.
For example, if you want to type 'কানন', you should type kan+CTRL+n. This is because if you simply type 'kann', the last two characters 'nn' will be interpreted as 'ণ' and you will see 'কাণ' instead of 'কানন' By pressing CTRL, the continuity between the two successive 'n'-s are broken and interpreted as separate characters.

*We recommend SolaimanLipi which was developed by Mr. Solaiman Karim at Ekushey.org and is available free. Visit their website at http://ekushey.org to download it. Windows users should put it in the 'C:\WINDOWS\Fonts' or 'C:\WINNT\Fonts' directory and Mac users should put it in '/Library/Fonts' directory.
