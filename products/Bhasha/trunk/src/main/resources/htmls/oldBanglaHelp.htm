Welcome users!

Using this software you can create really nice, styled HTML documents in Bangla. You need to install a Unicode Bangla Font. We recommend SolaimanLipi which was developed by Mr. Solaiman Karim at Ekushey.org and is available free. Visit their website to download it. Windows users should put it in the 'C:\WINDOWS\Fonts' or 'C:\WINNT\Fonts' directory and Mac users should put it in '/Library/Fonts' directory.

Look at the following example script:
আমার সোনার বাঙ্গলা , আমি তোমায় ভালবাশি
চির দিন তোমার আকাশ, তোমার বাতাশ,
আমার প্রাণে বাজায় বাঁশী ৷
- শ্রী রবীন্দ্রনাথ ঠাকুর

Bhasha Key Mapping

To change to Bangla: ALT+b
To change to Devnagari(Hindi): ALT+d
To change to English: ALT+e

স্বর বর্ণ গুলির ক্ষেত্রে, ছোট হাতের English হরফ, মানে small letters ব্যাবহার করা হয়েছে ৷
যেমন 'স' লিখতে হলে 's' type করুন ৷ আবার 'স্' লিখতে হলে 'S', মানে SHIFT টিপে 's' type করুন ৷ আপনি যদি 'SolaimanLipi.ttf' ব্যবহার করেন, তাহলে যুক্তাক্ষরের জন্য কোন অসুবিধা হবে না ৷
ব্যাঞ্জন বর্ণ গুলির ক্ষেত্রে, ছোট হাতের English হরফ মাত্রা গুলির জন্য ব্যবহার করা হয়েছে ৷
যেমন 'কি' লিখতে গেলে 'ki' type করুন ৷ 'কই' লিখতে গেলে 'kI' type করুন ৷

To break the continuity between two characters type: CTRL.
For example, if you want to type 'কানন', you should type kan+CTRL+n. This is because if you simply type 'kann', the last two characters 'nn' will be interpreted as 'ণ' and you will see 'কাণ' instead of 'কানন' By pressing CTRL, the continuity between the two successive 'n'-s are broken and interpreted as separate characters.
The contents can be saved in .bn files only.... When you generate HTML pages, and save it as, say xyz.html, then a backup file called xyz_HTMLBackup.bn is created in the same directory. If in future you want to modify that file xyz.html, 'Open' the backup .bn file and then edit.... HTML files cannot be edited directly.
